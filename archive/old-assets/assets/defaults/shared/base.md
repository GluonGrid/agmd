# Universal Agent Guidelines

This file contains universal guidelines that apply to all AI coding assistants across all projects.

## Core Principles

- Follow existing code conventions in the project
- Write clear, maintainable code
- Test before committing
- Ask questions when requirements are unclear

## Code Quality

- Keep functions small and focused
- Use descriptive names for variables and functions
- Add comments for complex logic
- Refactor when you see duplication

## Version Control

- Write clear commit messages
- Keep commits atomic and focused
- Review changes before committing

## Documentation

- Update documentation when behavior changes
- Keep README accurate and up-to-date
- Document non-obvious decisions

---

*This is the universal base configuration. Project-specific rules override these guidelines.*
